# Imágenes requeridas

Esta carpeta debe contener las siguientes imágenes:

## Imágenes del proyecto (hero)
- `hero-la-florida.jpg` - Imagen principal del proyecto (1600x1200px recomendado)
- `hero-la-florida.webp` - Versión WebP (opcional, para mejor rendimiento)
- `hero-la-florida.avif` - Versión AVIF (opcional, para mejor rendimiento)

## Imágenes del evento
- `evento.jpg` - Imagen de la sala del evento (1280x720px recomendado)
- `evento.webp` - Versión WebP (opcional)
- `evento.avif` - Versión AVIF (opcional)

**Nota**: Si solo tienes las versiones JPG, funcionará correctamente. Las versiones WebP y AVIF son opcionales para mejorar el rendimiento de carga.
