# Solución para error de deploy en Hostinger

## Problema
```
pull: error: The following untracked working tree files would be overwritten by merge:
	images/evento.jpg
	images/hero-la-florida.jpg
```

## Solución

### Opción 1: Modificar el script de deploy en Hostinger
Agregar antes del `git pull`:
```bash
git reset --hard HEAD
git clean -fd
git pull origin main
```

### Opción 2: Eliminar archivos manualmente en Hostinger
Conectarse por SSH o File Manager y eliminar:
- `images/evento.jpg`
- `images/hero-la-florida.jpg`

Luego ejecutar el deploy nuevamente.

### Opción 3: Forzar pull (si tienes acceso SSH)
```bash
cd /ruta/del/proyecto
git fetch origin
git reset --hard origin/main
```

## Nota
Los archivos `images/evento.jpg` y `images/hero-la-florida.jpg` ya están correctamente en el repositorio, por lo que después de resolver el conflicto, el deploy debería funcionar normalmente.
